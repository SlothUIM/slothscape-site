Frequently Asked Questions

What Java version do I need?

Java 8 or newer. Windows/macOS installers bundle a compatible runtime.

How do I report a bug?

Use the Contact page or open an issue on our GitHub repository. Include logs if available.

Is Discord RPC required?

No. It’s optional and off by default.

Where are settings stored?

By default in your user directory under .slothscape/. You can export/import profiles.
